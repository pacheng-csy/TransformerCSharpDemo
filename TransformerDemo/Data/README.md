# 三国演义问答数据格式说明

## 文件

- **sanguo_qa_train.jsonl**：训练集（建议约 800 条）
- **sanguo_qa_valid.jsonl**：验证集（建议约 200 条）

## 每行格式（JSONL）

每行一条 JSON 对象，UTF-8 编码，字段：

| 字段       | 类型   | 说明 |
|------------|--------|------|
| `question` | string | 问题，单句或短段中文问句 |
| `answer`   | string | 答案，1～3 句简洁回答 |

示例一行：

```json
{"question": "关羽在哪里被杀？", "answer": "关羽在荆州临沮被东吴吕蒙所害。"}
```

## 约定

- 问题、答案前后空白会被程序自动去除。
- 建议问题与答案均保持中短长度，便于小模型收敛（例如单句问、1～3 句答）。
- 内容可涵盖：人物生平与官职、战役事件、势力格局、时间线与因果关系、典故梗概等。
